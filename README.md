# SOUTHBEACHACQUISTIONS

Luxury real estate acquisitions — south coast, Dominican Republic. Where inventory meets investor capital.

This is a [Next.js](https://nextjs.org) 14 (App Router) project with Tailwind CSS, Framer Motion, and Lucide React.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deploy

Deploy on [Vercel](https://vercel.com) or any Node host. Run `npm run build` then `npm run start` for production.
